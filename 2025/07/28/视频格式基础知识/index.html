Ishamel Heathcliff

视频格式基础知识

封装格式

  • 封装格式
    • 包含视频、音频、章节、字幕等
    • 轨道
    • MKV与MP4
    • MKV可以封装FLAC音频和ASS/SSA字幕，MP4不支持
    • MP4兼容性好于MKV
    • MKV民间研发，代替AVI，兼容flac/ass等非工业标准格式
    • MP4权威机构研发，工业标准，代替MPG
    • 过时的RMVB/RM/AVI/MPG
  • 视频格式
    • H.264(8bit/10bit)=AVC=AVC1
    • H.265(8bit/10bit)=HEVC
    • RealVideo
      • RMVB/RM中常使用
    • VC-1
      • 微软主导
  • 音频格式
    • FLAC/ALAC/TrueHD/DTS-HD MA
    • AAC/MP3/AC3/DTS(Core)

基础参数

  • 分辨率
  • 帧率
    • 24000/1001=23.976
    • 30000/1001=29.970
    • 60000/1001=59.940
    • 25.000
    • 50.000
  • 码率
    • 视频文件体积/时间
    • Kbps、Mbps

图像表示方法

  • RGB模型
    • PNG、BMP
  • YUV模型
    • 亮度-色度模型(Luma-Chroma, Y-UV)
    • YCbCr(亮度-蓝色色度-红色色度)
    • JPEG
    • 几乎所有视频格式
  • 渲染
    • 将YUV转换为RGB

色深

  • 8bit与10bit
    • 8bit YUV转换后精度大概为8bit RGB的26%
    • 10bit YUV转换后大概为8bit RGB的97%
    • 主要表现在亮度较低的区域

色度半采样

  • YUV模型中Y比UV重要
    • Y以全分辨率记录，UV则可能减半甚至四分之一（色度半采样）
  • yuv420
    • YUV格式
    • 420采样
    • UV半采样
    • 亮度1920*1080
    • 色度 960*540
  • yuv444
    • 444采样，满分辨率
  • 片源、蓝光原盘等一般都是420采样
  • 将420拉伸为444是伪高清
    • 也可用于修复和处理

低频与高频

空间频率

  • 空间(spatial)指一帧图片内
  • 高频指亮度变化迅速的区域
  • 低频指亮度变化缓慢不明显的区域
  • 平面
    • 低频，亮度几乎没变化
  • 纹理
    • 高频，变化频繁，幅度有高有低
  • 线条
    • 高频，跳跃式突变
  • 或统称高频为线条，低频为非线条
  • 除了亮度平面，也适用于色度平面
  • “细节”一般指高频信息
    • 高频越多，信息量越大，编码运算量越大
    • 编码器码率空间分配
    • 对高频部分和低频部分的计算分配

时间频率

  • 时间(temporal)指帧与帧之间的变换
  • 动态
    • 帧与帧之间的变化强弱、变化频率
  • 动态高，编码运算量大
  • 人眼对高速变化场景敏感度低
    • 动态场景优先度低于静态场景
    • 编码器码率时间分配

清晰度与画质

  • 什么是清晰度？
    • 分辨率
    • 不代表清晰度
    • 决定清晰度的上限
    • 空间高频信息量
    • 原生、有效高频信息
    • 适度的人为增加高频信息
  • 什么是画质？
    • 空间方面的清晰度
    • 时间方面的动态效果
    • 视频画质的决定因素
    • 片源画质
    • BDRip优于TVRip
    • 播放条件
    • 播放硬件与软件
    • 码率
    • 时间与空间复杂度统称为编码复杂度
    • 复杂度高就需要更高的码率
    • 码率分配效率
    • H264优于RealVideo
    • 10bit优于8bit
    • 编码器与参数设置
    • 码率分配效率是对制作者的要求
    • 编码前预处理
    • 客观修复
    • 主观调整
    • 移除无效高频信息，降噪
